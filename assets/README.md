# Assets

Ce dossier doit contenir les icônes de l'application.

## Fichiers requis

Vous devez créer ou générer les fichiers suivants :

### Icônes

- `icon.png` - Icône principale (1024x1024px)
- `adaptive-icon.png` - Icône Android adaptative (1024x1024px)
- `splash.png` - Écran de démarrage (1242x2436px)
- `favicon.png` - Favicon web (48x48px)
- `notification-icon.png` - Icône de notification Android (96x96px)

## Thème Supernatural

Suggestions pour les icônes :

- **Symbole du pentagramme** (protection contre les démons)
- **Flammes orange/rouges** sur fond noir
- **Logo "SPN"** stylisé
- **Impala 67** (voiture emblématique)
- **Symbole anti-possession**

## Outils de génération

Vous pouvez utiliser :

1. **Canva** : https://www.canva.com (gratuit)
2. **Figma** : https://www.figma.com (gratuit)
3. **Icon Kitchen** : https://icon.kitchen (spécifique Expo/React Native)
4. **DALL-E / Midjourney** : Pour générer des icônes avec IA

## Template Canva suggéré

Couleurs :
- Background : #0D0D0D (noir)
- Accent 1 : #FF6600 (orange feu)
- Accent 2 : #8B0000 (rouge sang)

Police :
- Titre : Bold, style vintage/rock
- Idée : "SPN" ou pentagramme au centre

## Générer rapidement avec Expo

```bash
npx expo-icon-generator generate
```

Cette commande peut vous aider à générer les icônes dans les bonnes dimensions.

---

**Note** : Les icônes actuelles sont des placeholders. L'app fonctionnera mais utilisera les icônes par défaut d'Expo jusqu'à ce que vous les remplaciez.
